Please mind that Wicked Shell requires Eclipse 3.2 to work properly. There are a few ways to get Wicked Shell running on your Eclipse distribution.

  • The binaries/sources of the Wicked Shell are available here at Sourceforge on the Wicked Shell's download section. The file available contains the feature to be unpacked in your Eclipse installation directory. After restarting your Eclipse, you'll find two more views to your service.
  • You can also install Wicked Shell via the eclipse update mechanism. Create a remote update site at http://www.wickedshell.net/updatesite to install the Wicked Shell feature. The Eclipse will restart automatically afterwards.

If there are any problems with the installation, just give me a hint!
